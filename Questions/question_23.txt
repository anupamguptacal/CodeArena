Write a Program to enter an array and find the index of the second last occurence of the element in the array. If there is only one or no such occurence, return -1.
---
Input: [5,4,3,5,7,3,1], element = 5, size_of_array = 7
Output: [4,3,5,7,3,1], size_of_array = 6
---
Input: [6,6,6,7,6,7], element = 7, size_of_array = 6
Output: [6,6,6,6,7], size_of_array = 5
