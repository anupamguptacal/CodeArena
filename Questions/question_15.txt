Write a Program to delete the middle element of an array. In case of odd length arrays, delete the middle element, in case of even length arrays, delete the element that comes first when the array is read from the left hand side. 
---
Input: [5,7,1,8,5], size = 5
Output: [5,7,8,5], size = 4
---
Input: [5,7,1,3,8,5], size = 6
Output: [5,7,3,8,5], size = 5
